绘画工具

绘画工具是创作的基础。在绘画过程中，每种绘画工具可能都有不同的状态。例如，画刀和画笔有三个状态：清洁，脏，模式。

状态 描述 转变
清洁 绘画工具是干净的，没有着色。

当一个干净的工具在画板表面绘制和其它的颜料交互时，这个工具便成了脏的。

当“自动重置”功能打开时，这个功能会清理当前保存的模式，这种情况下，这个工具就有“自动清理”的效果。

脏 绘画工具在绘制过程中会一直保持相同的颜色。

一个脏的工具可以通过“清洁”按钮返回到清洁状态。如果“自动重置”选项被禁用，那么这个工具在整个绘制过程中都保持脏的状态。

如果“自动复位”选项已启用，则每次绘制完成后都会返回到“模式”状态。如果“模式”状态没有颜色值，则变为“清洁”状态。

一个临时的脏状态也可以保存为一个“模式”，以便以后可以重复使用。

模式 绘画工具拥有相同的颜色，湿度，厚度等信息。

画笔和画刀在绘制过程中都可能变脏，我们将这种有颜色的脏的状态称为“模式”状态，此状态可以保存，然后通过“自动重置”选项重复使用。

这个状态可以通过“清洁”按钮来恢复到清洁状态。

 

颜料管工具是一种最简单的绘画工具，它不会与画布进行双向交互，使用它的目的是将一些颜料直接涂抹在画布上。因此，“自动重置”和“自动清洁”选项对于颜料管工具来说没有任何意义。

“自动重置”选项和“自动清洁”是某些绘画工具的特性，但是这两个选项不能一起使用，只有其中一个可以启用。

当启用“自动重置”选项时，以下情况中的模式会被自动保存：

  1. 从调色板的颜料碗中蘸取颜料时
  2. 使用颜色选取或模式选取工具时
  3. 按下“清洁”按钮时

颜料管

颜料管只是用来提供各种各样的颜料。Apple Pencil的压力控制着颜料的多少，压力越大，挤出的颜料也就越多，越厚；Apple Pencil的高度和方位对颜料管没有任何影响。

画笔

画笔是油画工具中的重要组成部分，它的形状不同，绘画的效果也各不相同。最常见的画笔有：平板刷，圆头刷和扇形刷等。每个画笔的细节将在下面的段落中描述。

所有这些画笔都有着共同的行为和控制。与“画刀”相比，“画笔”在绘画中始终留下“毛茸茸”痕迹，“画笔”的刷毛会根据你的力道进行收缩和扩展。与画刀相同，压力对颜料的厚度也起着重要的作用。厚度与压力成反比：压力越大，绘制的层就会更薄。

画刀

用画刀作画和画笔作画的感觉是完全不同的，画刀能达到用画笔难以达到的效果。我们的APP更是很好的利用了这一点，并且在厚涂，刮色等方面很好的模拟了实际画刀的特点。当然了，不同形状的画刀，画出的效果也截然不同。

  • 短刀片会产生有角度的线条。
  • 长刀片可以很容易地大面积涂色。
  • 锋利的刀片可以用来修理画布上凸起的线头，清除画面的小颗粒或污点。

毫无疑问地说，画刀是艺术家们使用最多的绘画工具之一。

在我们的工具中，画刀也能够通过Apple Pencil来很好的控制：

Apple Pencil 画刀描述
压力

控制颜料的厚度。压力越大，画板上获得的图层就越薄。画刀的压力不收画刀的大小和形式的影响。

高度

控制刀的垂直方向的角度。当高度角小于30°时，画刀完全放置在表面上。这个角度可通过Apple Pencil控制在30°和90°之间。

方位 在绘画过程中控制画刀的方向。

画刀轴上的旋转能够提供很多功能。不幸的是，Apple Pencil还不支持这些。与画笔相同，画刀的旋转由Joystick来控制。

画刀作画

一般来说，画刀在绘画中有两个角色：混合颜色和去除颜色。这取决于画刀的方向，方向和旋转。

当画刀旋转时，即画刀与颜料表面成角度，沿着画刀的运动方向就会有不同的绘画效果。画刀向角度小于90°的方向移动时，处于混合模式，称为前向绘画；当它向相反方向（角度大于90°）移动时，这是一种清洁模式，称为“后向绘画”。在这种模式下，表面上的颜料会减少。

在下面的屏幕截图中，画刀的旋转是45°。向左移动是清洁，向右移动的是正常的绘画模式。

刮画效果

在清洁模式下，当画刀方向接近90°或旋转接近90°或-90°时，会产生刮画效果，而且这种效果跟压力息息相关。

Pick Up Gesture

In general, when we paint slowly, most of color will leave on the surface. But some time, we need to pick up the color from painting surface. Typically, it is the purpose of Palette. A gesture is provided to meet this need. This gesture consists of the following steps:

  1. Select a more thick color region
  2. Press you knife hardly on paint surface
  3. And then, rise it up very quickly

It performs like a Tap. The most important is to leave the surface as quickly as possible.

Utility Tools

Color Filling

This tool is used to select a Color on painting support. It isn't a really Painting tool. It simulates the action to fill the current painting tool with Color such as Tube, Knife, Brush, etc. This action is similar as you dip your painting tool in color liquid.

Drawing Tool Description
Tube

Select a color of tube

The quantity of filling is independent of 压力 of Apple Pencil when you click on Painting support or color icon. It is maximum.

Knife

Select a color and fill the current knife with this color liquid.

The quantity of filling depends on the 压力 of Apple Pencil when you click on Painting support or color icon

Brush

Select a color and fill the current brush with this color liquid.

The quantity of filling depends on the 压力 of Apple Pencil when you click on Painting support or color icon

When you select the Color Picker, the current drawing tool gets disabled. It will be enabled after the selection operation.

The color selected is a pure color without any light and shadow influence by the 3D display engine.

Color Pattern Picker

This tool selects a color pattern using the Pickup Gesture. The current painting tool gets clean before the color pattern pickup.

Drawing Tool Description
Knife

Select a color pattern for the current knife.

The quantity of filling depends on the 压力 of Apple Pencil when you click on Painting support.

Brush

Select a color pattern for the current brush.

The quantity of filling depends on the 压力 of Apple Pencil when you click on Painting support.

Same as Color Filling, the color selected is a pure color without any light and shadow influence by the 3D display engine.

Dryer

Dryer is a utility to control the dry state of color on canvas. You can use it make some regions dry. When you click on this tool, the current canvas will be overlaid by a layer. The wet region is displayed in white semi-transparent layer. The dried region is totally opaque and white. You can place the dryer anywhere to change it.

Once it is done, this dried region becomes rigid surface: the color in this region can be changed: neither be blended, non be removed.

The dryness of color can give some special effect with Knifes. You can remove the wet colors in erase mode to show the color under layer.

This tool can be used only on the canvas. It cannot be used on Palettes.

Wetter

The user interface of this tool is similar to Dryer. When you click on this tool, the current canvas will be overlaid by a thin layer. The wetness is displayed in black semi-transparent color. The degree of black represents wetness.

Turpentine

This tool is used to dilute colors to thinner the layer and spread the liquid. It modifies the Wetness of color on Painting support.

Palette

Palette is an extension of main Canvas. We use it to prepare colors and patterns of Painting Tools. It shares Painting tools with main Canvas.

Automatic Save

There are two parameters in this App to enable the Automatic Save. One parameter is to enable/disable this feature. Another is to set the interval of each save.

In the editor, if this feature is enabled and after the each stroke, a timer is created with the interval parameter defined in the App. The action of save will be invoked after the interval time.

Sharing, Import & Export

A painting can be Imported and exported from/to image in png format.

Import

There are two mechanisms to import an Image:

  • From a Workshop, an image can be imported on top of the current Painting content. The content before the import will be put in Dry.
  • From outside of Impasto such as App Photo, you can select an image and then creates a new Painting in the same size of imported image and imports the image. In fact, the import consists of two steps:
    • Create a temporary file in a shared folder
    • Create an Painting inside of Impasto, which can occurs on the startup or in the Gallery when you scroll.

Please note that when you import a lot of images, the next startup of Impasto can be slow down.

Export

Inside of Workshop, the button "Export" converts the current Painting only in Image and then saves it in Photo Library.

Share

This feature converts the current Painting in Image and then publishes it in any social network you have such as Facebook, WeChart etc.

Undo/Redo

Due to the limitation on memory of IPad, the only 20 undo/redo operations is supported. If you can more undo operations, you need to use the Timeline to go back to anytime.
